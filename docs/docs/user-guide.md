# CDS User Guide

After successfully deploying CDS, you can interact with the system through three primary methods. Most users will use a combination of these methods depending on their workflow.

## Interaction Methods

CDS provides three ways to interact with the system:

1. **Web User Interface (UI)** - Interactive visual interface for search and exploration
2. **Command Line Interface (CLI)** - Built on top of the REST API, offering a simplified interface for common tasks including collection management, ingestion, and search
3. **REST API** - Direct programmatic access via HTTP/curl for custom integrations
